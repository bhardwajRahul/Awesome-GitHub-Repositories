## Contribution Guidelines

We love contributions! Here's how to help keep this list awesome:

1. Fork the repository
2. Add new repositories (follow the existing format)
3. Update star counts if significantly changed
4. Keep one repository per line
5. Add a short, clear description
6. Place in the correct category (or create a new one if needed)
7. Open a Pull Request with a clear title like "Add: Cool New Repo for Kubernetes"
